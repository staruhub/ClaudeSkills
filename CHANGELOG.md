## [1.0.0] - 2025-11-13

### Added
- Initial release of ClaudeSkills collection
- **Core Skills** (8 professional skills):
  - Code Review Expert - Professional code review with 8 comprehensive dimensions
  - Coze API Integration - Complete guide for ByteDance Coze platform
  - CrewAI Developer - Multi-agent AI systems development
  - Document Skills (Official) - DOCX, PDF, PPTX, XLSX processing
  - Flutter API - Comprehensive Flutter development reference
  - MCP Builder (Official) - Model Context Protocol server development
  - Podcast Generator - AI podcast generation using Volcano Engine
  - WeChat Article Writer - Professional WeChat official account content creation
- **Additional Skills** (5 standalone skills):
  - C Drive Cleaner Skill
  - Seedream Image Generation
  - 高考专家技能开发
  - 高考命题专家技能
- Comprehensive documentation in English and Chinese
- Example prompts for coding and writing tasks
- Contributing guidelines (CONTRIBUTING.md, CONTRIBUTING.zh-CN.md)
- Project structure following GitHub best practices
- Standard configuration files (.gitignore, requirements.txt, pyproject.toml)
- CI/CD workflows for automated testing
- Individual README files for each skill module

### Changed
- Reorganized project structure following GitHub best practices
- Updated all documentation to November 2025
- Unified skill package format and directory structure
- Moved test files to dedicated `tests/` directory
- Improved dependency management with pyproject.toml

### Fixed
- Removed duplicate archive files (.zip)
- Cleaned up system-generated files (__MACOSX)
- Standardized directory structure across all skills
- Updated outdated documentation dates

### Removed
- Duplicate .zip archive files
- System-generated __MACOSX directories
- Redundant configuration files
