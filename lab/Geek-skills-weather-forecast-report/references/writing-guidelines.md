# 省级自然科学基金专题研究报告撰写指南

## 一、总体原则

### 1.1 科学性原则
- 数据真实可靠，来源清晰
- 方法科学规范，可复现
- 结论有据可依，不夸大

### 1.2 创新性原则
- 明确指出创新点
- 与现有研究对比突出差异
- 创新应有实验验证支撑

### 1.3 规范性原则
- 术语使用规范统一
- 格式符合基金要求
- 引用标注准确完整

### 1.4 完整性原则
- 逻辑链条完整闭环
- 各章节内容衔接紧密
- 图表与正文呼应

## 二、各部分撰写要点

### 2.1 摘要撰写

**结构模板（中文）**：
```
[研究背景]随着/鉴于...[领域]对[需求]的迫切需求，[问题]成为亟需解决的关键问题。
[研究目的]本研究旨在...
[研究方法]采用...方法，通过...，构建了...
[主要结果]实验结果表明：(1)...；(2)...；(3)...
[结论]研究证明了...，为...提供了...
```

**注意事项**：
- 避免使用"本文"开头，可用"本研究"
- 避免出现图表编号引用
- 数据要具体，避免笼统描述
- 不要出现参考文献引用

### 2.2 研究背景撰写

**撰写逻辑**：
1. 从宏观到微观：行业需求 → 技术领域 → 具体问题
2. 点明痛点：现有方法的不足
3. 引出本研究的必要性

**常用句式**：
- "...是...的关键/核心/基础"
- "然而，现有方法存在...局限"
- "因此，亟需发展..."
- "本研究聚焦于..."

**文献综述要点**：
- 按时间线或技术路线组织
- 既要肯定贡献，也要指出不足
- 最后导向本研究的切入点

### 2.3 研究方法撰写

**技术路线图要求**：
- 清晰展示研究流程
- 包含输入、处理、输出
- 标注关键技术/方法
- 体现逻辑层次

**方法描述要点**：
- 先说"做什么"，再说"怎么做"
- 公式要有编号和说明
- 参数设置要给出依据
- 算法流程可用伪代码

### 2.4 实验结果撰写

**结果呈现原则**：
- 先总体后细节
- 先主要后次要
- 先成功后失败

**图表使用规范**：
- 表格：适合精确数值对比
- 折线图：适合趋势展示
- 柱状图：适合分类对比
- 热力图：适合相关性分析

**结果分析要点**：
- 不只是描述数据，要解释原因
- 与预期对比，分析差异
- 与文献对比，突出优势
- 承认局限，分析原因

### 2.5 结论撰写

**结论要点**：
- 总结性而非重复性
- 对应研究目标逐条回应
- 突出创新贡献
- 指出局限和展望

**常见问题**：
- ❌ 简单重复摘要内容
- ❌ 引入新数据或论点
- ❌ 过度夸大研究意义
- ✅ 概括提炼，升华主题

## 三、学术表达规范

### 3.1 时态使用
- 研究背景：一般现在时
- 研究方法：一般过去时
- 实验结果：一般过去时
- 结论：一般现在时

### 3.2 人称使用
- 推荐使用被动语态或"本研究"
- 避免"我们"、"笔者"等
- 英文摘要避免第一人称

### 3.3 数字规范
- 数据保留适当有效数字
- 百分比不超过小数点后一位
- 范围用"~"连接，如"85%~90%"
- 小于1的小数需写"0"，如"0.85"

### 3.4 术语规范
- 首次出现的缩写需给出全称
- 全文术语保持一致
- 专业术语使用规范写法

## 四、天气预报领域特定要求

### 4.1 数据描述规范
- 明确数据来源（气象站/再分析资料/数值模式）
- 说明时间分辨率和空间分辨率
- 描述质控标准和方法

### 4.2 评估指标选择

**连续变量（温度、风速等）**：
- 平均绝对误差（MAE）
- 均方根误差（RMSE）
- 相关系数（CC）
- 技巧评分（SS）

**分类变量（有无降水等）**：
- 准确率（Accuracy）
- TS评分（Threat Score）
- ETS评分（Equitable Threat Score）
- 空报率（FAR）
- 漏报率（POD）

### 4.3 基准对比
- 必须与传统方法对比
- 建议与机器学习经典方法对比
- 可与业务系统预报对比

### 4.4 案例选择
- 选择具有代表性的天气过程
- 覆盖不同类型/季节/强度
- 成功和失败案例都要分析

## 五、常见问题与解决

### 5.1 内容问题
| 问题 | 解决方案 |
|------|---------|
| 背景与研究脱节 | 明确"因此本研究..."的逻辑链 |
| 方法描述不清 | 添加流程图和伪代码 |
| 结果缺乏分析 | 每组数据后加原因解释 |
| 结论过于笼统 | 对应目标逐条总结 |

### 5.2 格式问题
| 问题 | 解决方案 |
|------|---------|
| 图表无编号 | 统一添加"图1"、"表1" |
| 参考文献不规范 | 使用GB/T 7714标准 |
| 公式无编号 | 右对齐添加(1)、(2) |
| 段落层次混乱 | 使用规范序号体系 |

### 5.3 语言问题
| 问题 | 解决方案 |
|------|---------|
| 口语化表达 | 改用书面语 |
| 术语不一致 | 建立术语表统一 |
| 中英混杂 | 专有名词保留英文，其余用中文 |
| 长句难懂 | 拆分为短句 |
