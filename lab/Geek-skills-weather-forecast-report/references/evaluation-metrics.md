# 气象预报评估指标说明

## 一、连续变量评估指标

### 1.1 平均绝对误差（MAE）

**定义**：预测值与实际值之差的绝对值的平均值

**公式**：
```
MAE = (1/n) × Σ|yᵢ - ŷᵢ|
```

**特点**：
- 单位与原始数据相同
- 对异常值不敏感
- 越小越好

**适用场景**：温度、湿度、气压等连续变量

### 1.2 均方根误差（RMSE）

**定义**：预测值与实际值之差的平方的平均值的平方根

**公式**：
```
RMSE = √[(1/n) × Σ(yᵢ - ŷᵢ)²]
```

**特点**：
- 单位与原始数据相同
- 对大误差更敏感
- 越小越好

**适用场景**：风速、降水量等对大误差敏感的变量

### 1.3 相关系数（CC）

**定义**：预测序列与实际序列的线性相关程度

**公式**：
```
CC = Σ[(yᵢ - ȳ)(ŷᵢ - ȳ̂)] / √[Σ(yᵢ - ȳ)² × Σ(ŷᵢ - ȳ̂)²]
```

**特点**：
- 取值范围[-1, 1]
- 1表示完全正相关
- 越接近1越好

**适用场景**：评估预报趋势准确性

### 1.4 技巧评分（SS）

**定义**：相对于参考预报（如气候平均）的改进程度

**公式**：
```
SS = 1 - (MSE_forecast / MSE_reference) × 100%
```

**特点**：
- 可为负值（比参考预报差）
- 100%表示完美预报
- 越大越好

**适用场景**：评估预报方法相对于基准的提升

### 1.5 平均偏差（BIAS）

**定义**：预测值与实际值之差的平均值

**公式**：
```
BIAS = (1/n) × Σ(ŷᵢ - yᵢ)
```

**特点**：
- 正值表示系统性高估
- 负值表示系统性低估
- 越接近0越好

**适用场景**：评估系统性偏差

## 二、分类变量评估指标

### 2.1 列联表

| | 实际有（O=1） | 实际无（O=0） |
|---|---|---|
| 预报有（F=1） | 命中(a) | 空报(b) |
| 预报无（F=0） | 漏报(c) | 正确否定(d) |

### 2.2 准确率（Accuracy）

**定义**：正确预报占总预报的比例

**公式**：
```
Accuracy = (a + d) / (a + b + c + d)
```

**特点**：
- 取值范围[0, 1]
- 对不平衡样本敏感
- 越大越好

### 2.3 命中率/检测概率（POD）

**定义**：正确预报事件发生占实际发生的比例

**公式**：
```
POD = a / (a + c)
```

**特点**：
- 取值范围[0, 1]
- 评估对事件的捕捉能力
- 越大越好

### 2.4 空报率（FAR）

**定义**：错误预报事件发生占所有预报发生的比例

**公式**：
```
FAR = b / (a + b)
```

**特点**：
- 取值范围[0, 1]
- 评估预报的可信度
- 越小越好

### 2.5 TS评分/临界成功指数（CSI）

**定义**：命中次数占总有效预报的比例

**公式**：
```
TS = a / (a + b + c)
```

**特点**：
- 取值范围[0, 1]
- 综合考虑命中、空报、漏报
- 越大越好

### 2.6 ETS评分

**定义**：扣除随机命中后的技巧评分

**公式**：
```
a_random = (a + c)(a + b) / (a + b + c + d)
ETS = (a - a_random) / (a + b + c - a_random)
```

**特点**：
- 取值范围[-1/3, 1]
- 消除了随机预报的影响
- 越大越好

### 2.7 Heidke技巧评分（HSS）

**定义**：相对于随机预报的改进程度

**公式**：
```
HSS = 2(ad - bc) / [(a+c)(c+d) + (a+b)(b+d)]
```

**特点**：
- 取值范围[-1, 1]
- 0表示与随机预报相当
- 越大越好

## 三、概率预报评估指标

### 3.1 Brier评分（BS）

**定义**：概率预报与实际结果的均方误差

**公式**：
```
BS = (1/n) × Σ(pᵢ - oᵢ)²
```

**特点**：
- 取值范围[0, 1]
- 0表示完美预报
- 越小越好

### 3.2 Brier技巧评分（BSS）

**定义**：相对于气候概率预报的改进

**公式**：
```
BSS = 1 - BS / BS_climatology
```

**特点**：
- 可为负值
- 1表示完美预报
- 越大越好

### 3.3 可靠性图

**作用**：评估概率预报的可靠性

**解读**：
- 理想情况下，预报概率=实际频率
- 在可靠性图上应贴近对角线

## 六、指标计算代码示例

```python
import numpy as np
from sklearn.metrics import mean_absolute_error, mean_squared_error

def evaluate_continuous(y_true, y_pred):
    """连续变量评估"""
    mae = mean_absolute_error(y_true, y_pred)
    rmse = np.sqrt(mean_squared_error(y_true, y_pred))
    cc = np.corrcoef(y_true, y_pred)[0, 1]
    bias = np.mean(y_pred - y_true)
    return {'MAE': mae, 'RMSE': rmse, 'CC': cc, 'BIAS': bias}

def evaluate_categorical(y_true, y_pred):
    """分类变量评估"""
    a = np.sum((y_true == 1) & (y_pred == 1))  # 命中
    b = np.sum((y_true == 0) & (y_pred == 1))  # 空报
    c = np.sum((y_true == 1) & (y_pred == 0))  # 漏报
    d = np.sum((y_true == 0) & (y_pred == 0))  # 正确否定
    
    pod = a / (a + c) if (a + c) > 0 else 0
    far = b / (a + b) if (a + b) > 0 else 0
    ts = a / (a + b + c) if (a + b + c) > 0 else 0
    
    a_random = (a + c) * (a + b) / (a + b + c + d)
    ets = (a - a_random) / (a + b + c - a_random) if (a + b + c - a_random) > 0 else 0
    
    return {'POD': pod, 'FAR': far, 'TS': ts, 'ETS': ets}
```
