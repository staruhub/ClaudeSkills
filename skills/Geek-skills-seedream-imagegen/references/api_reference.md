# Seedream 4.0 API 参考文档

## API 概述

Seedream 4.0 是 ByteDance 开发的下一代图像生成模型,通过 Segmind API 提供服务。该模型支持文本到图像生成、图像编辑和批量生成,具有高分辨率(最高 4K)、优秀的文本渲染能力和专业级视觉质量。

**API 端点**: `https://api.segmind.com/v1/seedream-4`

**认证方式**: API Key (通过 `x-api-key` 请求头传递)

**定价**: 约 $0.027-0.035 每张图片

## 请求参数

### 必需参数

| 参数 | 类型 | 描述 |
|------|------|------|
| `prompt` | string | 图像描述文本。清晰描述主题、风格、构图等元素 |

### 可选参数

#### 尺寸和分辨率

| 参数 | 类型 | 默认值 | 描述 |
|------|------|--------|------|
| `size` | string | "2K" | 图像分辨率。可选值: "2K" (2048x2048), "4K" (4096x4096), "custom" |
| `width` | integer | 2048 | 自定义宽度 (当 size="custom" 时)。范围: 1024-4096 |
| `height` | integer | 2048 | 自定义高度 (当 size="custom" 时)。范围: 1024-4096 |
| `aspect_ratio` | string | "1:1" | 纵横比。可选值: "1:1", "16:9", "9:16", "4:3", "3:2", "3:4", "2:3", "21:9" |

#### 生成选项

| 参数 | 类型 | 默认值 | 描述 |
|------|------|--------|------|
| `max_images` | integer | 1 | 生成图像数量。范围: 1-15 |
| `sequential_image_generation` | string | "disabled" | 批量生成控制。可选值: "auto" (顺序生成), "disabled" (独立生成) |

#### 参考图像

| 参数 | 类型 | 默认值 | 描述 |
|------|------|--------|------|
| `image_input` | array[string] | null | 参考图像 URL 列表,用于指导生成风格和构图。最多 3 个 URL |

## 使用建议

### 提示词最佳实践

1. **结构化描述**: 按照"主题 + 风格 + 细节"的顺序组织提示词

2. **文本渲染**: Seedream 4.0 擅长文本渲染,可以在提示词中包含文字要求

3. **多面板布局**: 支持网格布局和系列设计

### 尺寸选择指南

| 用途 | 推荐设置 |
|------|----------|
| 社交媒体图片 | size="2K", aspect_ratio="1:1" 或 "9:16" |
| 网页横幅 | size="2K", aspect_ratio="16:9" 或 "21:9" |
| 打印海报 | size="4K", aspect_ratio="4:3" 或 custom |
| 产品图片 | size="2K", aspect_ratio="3:2" 或 "4:3" |
| 手机壁纸 | size="2K", aspect_ratio="9:16" |
| 桌面壁纸 | size="4K", aspect_ratio="16:9" |

### 批量生成策略

#### 独立变体 (sequential="disabled")
- 适用于: 需要多个独立设计方案
- 特点: 每张图像完全独立

#### 顺序系列 (sequential="auto")
- 适用于: 需要主题一致的系列图像
- 特点: 保持统一的风格、色调和构图

### 参考图像使用

参考图像可用于:
- **风格指导**: 提供目标艺术风格的示例
- **品牌一致性**: 上传品牌颜色、logo 作为参考
- **构图参考**: 指导图像布局和元素位置

最佳实践:
- 使用高质量图像 (PNG, JPEG)
- 每个请求最多 3 张参考图
- 参考图应该清晰展示你想要的特定元素

## 性能与限制

- **生成时间**: 2K 图像约 1.8 秒,4K 图像约 4-6 秒
- **最大分辨率**: 4096 x 4096 像素
- **批量限制**: 单次请求最多 15 张图像
- **参考图限制**: 最多 3 张参考图像
