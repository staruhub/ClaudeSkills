### Theme Object 映射规则

```
palette[0] → theme.primary  (最深，用于标题和深色背景)
palette[1] → theme.secondary (深辅助，正文)
palette[2] → theme.accent    (中间色调强调)
palette[3] → theme.light     (浅色辅助)
palette[4] → theme.bg        (背景色)
```

> ⚠️ 如果色板中最深色不适合做 primary (如 Soft & Creative)，
> 手动调整映射，确保 primary 是最深色、bg 是最浅色。
> 文字与背景对比度必须 ≥ 4.5:1 (WCAG AA)。

| 元素 | 字重 | 说明 |
|------|------|------|
| 主标题 | **bold** | 唯一允许加粗的地方 |
| 副标题 | bold | 可选 |
| 正文 | normal | **正文严禁加粗** |
| 图例/脚注 | normal | 严禁加粗 |

### 字号层级 (Mode A: PPTX)

| 元素 | 推荐字号 | 倍率 |
|------|---------|------|
| 封面主标题 | 48-72pt | 3-5x |
| 封面副标题 | 24-36pt | 1.5-2x |
| 章节标题 | 36-48pt | 2.5-3x |
| 内容标题 | 28-36pt | 2x |
| 正文 | 14-18pt | 1x (base) |
| 图例/脚注 | 10-12pt | 0.6-0.7x |

> **铁律**: 相邻层级字号差 ≥ 20%。标题至少是正文的 2 倍。

## Style Recipes

四种预定义风格，选一种贯穿全部页面。

### Sharp — 锐利商务
- 直角 (rectRadius: 0)
- 实线边框
- 高对比配色
- 适合: 金融、咨询、政府

### Soft — 柔和通用
- 微圆角 (rectRadius: 0.04-0.08)
- 淡色阴影 (shadow: { type:"outer", blur:4, offset:2, opacity:0.1 })
- 适合: 教育、培训、通用商务

### Rounded — 友好圆润
- 大圆角 (rectRadius: 0.12-0.2)
- 无边框，色块区分
- 适合: 消费品、社交、年轻受众

### Pill — 胶囊现代
- 全圆角 (rectRadius: 0.5 on short side)
- 标签式布局
- 适合: SaaS、科技产品、创意

## 间距系统 (Spacing)

### PPTX 安全边距

```
内容区域: x ≥ 0.5", y ≥ 0.5"
右边界: x + w ≤ 9.5"
下边界: y + h ≤ 5.1" (为页码徽章留空间)
```

### 间距基数

| 元素间关系 | 间距 |
|-----------|------|
| 标题→正文 | 0.3-0.4" |
| 段落间 | 0.2-0.3" |
| 图标→标签 | 0.1-0.15" |
| 卡片间 | 0.2-0.3" |
| 边框→内容 | 0.15-0.2" |

## 信息图风格库 (Mode B)

### 风格 1: Notion 手绘风 (默认)

**风格前缀:**
```
Notion官方插画风格松弛感线稿信息图。线条特征：纯黄色为主，搭配其他适合的颜色。
线条粗细不均匀，像马克笔随手画的质感。笔触松弛、略带抖动，不追求工整。
所有图标、人物、图表都保持手绘涂鸦感。少部份地方填充黑色让整体压实。
人物画法：简笔画风格女孩，短发，圆润的头、点或线表示五官，
肢体夸张、动作松弛自然，可以是完整人物或只露出局部。
```

**风格后缀:**
```
禁止事项：不要彩色渐变或复杂配色。不要粗黑边框或生硬分隔线。
不要3D效果、阴影、立体感。不要密集文字堆砌，文字内容简化。
保持大量留白和呼吸感。装饰元素要克制。图片比例16:9，中文标注。
```

### 风格 2: 极简商务风

**风格前缀:**
```
极简主义商务信息图。纯色背景（浅灰或白色），几何形状分区。
使用扁平图标和简洁线条。配色限制在3色以内。
大量留白，信息密度低。字体清晰现代。无装饰性元素。
```

### 风格 3: 深色科技风

**风格前缀:**
```
深色主题科技信息图。背景为深蓝或纯黑，内容用霓虹色（青、绿、橙）高亮。
线条纤细发光。数据以发光面板形式展示。图标为线性图标带微光效果。
整体氛围：未来感、专业、高端。
```

### 风格 4: 莫兰迪柔和风

**风格前缀:**
```
莫兰迪色系柔和信息图。低饱和度配色，如雾粉、灰绿、淡紫、米白。
元素带有柔和阴影和半透明叠加。排版优雅，行距宽松。
适合高端品牌和女性受众。整体氛围：精致、宁静、高级感。
```

### 内容描述写作指南

好的内容描述 = 视觉化 + 有层次 + 有数据 + 有动作 + 有标注

```
✅ 好: 一个困惑的人被漂浮的AI热词包围，如"提示词过时了"和"程序员被淘汰"，
头顶飘着问号。两侧有喇叭在广播矛盾信息。底部数据："72%开发者已使用AI编码"。

❌ 差: 画一张关于AI编程的图。
```
