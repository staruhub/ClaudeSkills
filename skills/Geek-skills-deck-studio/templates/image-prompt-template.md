# Image Prompt 模板（模式 C，每页从 page brief 组装）

组装规则与引擎适配见 `../references/image-branch.md`。

```
为演示文稿生成第 {page_no} 页，{page_type} 页面，比例 {16:9|3:4}。

【风格】{风格名}：{style brief 原文粘贴}

【构图】{visual_structure 展开为空间描述，如：左侧 40% 为文字区，
右侧 60% 为图表区，页面上方 1/5 为标题带}

【标题】原样渲染："{title}"，超粗黑体，页面最高视觉权重
【正文】{≤30 字原文} —— 或：正文区留白，只画版式框架（留位策略）
【图像/图表】{image_need 展开：图表类型与数据形态描述}

【留白与层级】页边距不小于画面宽度 5%；视觉层级：标题 > {关键元素} > 正文
【排版】中文横排，标点不出现在行首；中英混排时英文占比不超过 20%

禁止：文字重叠、文字压在复杂图像上、伪汉字乱码、逐字竖排英文、
超过两种字体、页面四边贴字无留白、装饰元素盖过标题、每页超过 5 个信息块。
```

批量顺序：封面 → 抽 1 页正文验证风格 → 用户认可 → 其余页。每 5 页与封面比对一次一致性。
