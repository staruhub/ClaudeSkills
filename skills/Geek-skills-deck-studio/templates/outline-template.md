# Outline 模板（Step 4 / 模式 A 交付格式）

```markdown
# [Deck 标题]

- **主题**：一句话
- **受众**：谁看 + 他们最关心什么
- **目标**：看完后希望对方做什么/记住什么
- **任务类型**：汇报|路演|培训|提案|复盘|咨询报告|招聘介绍|自媒体分享
- **推荐风格**：[风格名]（理由一句话）
- **页数建议**：N 页
- **交付模式**：A outline / B 内容稿 / C 视觉图

## 页面规划

| # | 页面类型 | 标题 | 一句话摘要 |
|---|---------|------|-----------|
| 1 | cover | ... | ... |
| 2 | toc | ... | ... |
| ... | | | |

## 待确认

- [大纲层面需要用户拍板的点，如：是否包含竞品对比章节]
```

规则：标题写结论不写话题；摘要一句话必须回答"这页凭什么存在"；给出待确认点后**停下等确认**。
