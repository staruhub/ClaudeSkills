## 设计要点（构成主义硬约束）

- 三色封闭：革命红 `#D32F2F` / 纯黑 `#111` / 纸白 `#F5F1E8`，**单一强调色**
- 母题必须贯穿：红楔 / 对角线标题 / 超大标点 / 黑圆——首尾(p1/p9)对角爆发，
  中段(p4 红底、p7 黑底)做明暗节奏，但每页至少一处对角元素接线
- 强调词用**黑底白字锐块**反白，不用低对比色字（p4 教训）
- 数据/流程页用**巨型衬底数字**（墨色极淡）建立纵深、消化死区（p6 教训）

## 本例沉淀的教训（评审两轮抓出）

1. **B4 死区**：流程/列表页下半留白 → 巨型衬底数字 + 第二行 + 对角楔填满
2. **B7 单一强调色**：修 A 页时别引入 B 页被批的柔弱色（淡粉），红-粉叠层互相打架
3. **网格 vs 母题**：装饰母题（红楔）不得压过页脚/页码系统——"母题赢了，网格纪律
   不能输"，收在页脚线以上
4. 对角母题若用 tone-on-tone 会太含蓄第一眼不可见，关键页要用实色对撞
