# 清风讲堂

- **一句话定位**：水蓝清爽、低压迫感，让学员长时间看不累的课件风
- **适用场景**：培训课件、知识分享、入职引导、操作手册化演示
- **不适用场景**：融资路演、高端提案（气质太"温"）
- **色板**：`#F7FAFC`（浅底）/ `#2B6CB0`（主蓝）/ `#1A365D`（深蓝标题）/ `#ED8936`（强调橙，仅重点）/ `#4A5568`（正文灰）
- **字体气质**：思源黑体 Regular 为主，标题 Medium；亲和不失专业
- **版式特征**：顶部色条+章节标签；卡片化知识块；步骤编号醒目；每章固定"小结页"版式
- **图像倾向**：扁平插画 > 照片；图标辅助记忆点；示意图多于装饰图
- **页面密度**：中；允许比商务风多一点内容，但仍守 3-5 要点
- **标题语气**：引导式，"三步搞定环境配置"式
- **image2 适配**：适合（扁平插画风 image 模型表现稳定）
- **直接导出 PPT**：非常适合（培训场景常需讲师后期改字）

## Style Brief（注入生成器）

清爽教学风格：极浅蓝灰背景，深蓝标题配主蓝色条与章节标签，知识点用白色圆角卡片承载，扁平风格插画与线性图标辅助说明，橙色仅标注最重要的记忆点，整体明亮、友好、有呼吸感，适合长时间阅读。
