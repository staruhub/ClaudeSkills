# 学术蓝灰

- **一句话定位**：论文答辩与学术报告的稳妥选择，严谨不寒酸
- **适用场景**：论文答辩、学术会议报告、研究进展汇报、课题申请
- **不适用场景**：商业路演、自媒体内容、创意提案
- **色板**：`#FCFCFD`（底）/ `#2C3E50`（深蓝灰标题）/ `#5D6D7E`（正文）/ `#2E86C1`（引用与链接蓝）/ `#E8EBEE`（图表底）
- **字体气质**：中文思源宋体标题 + 思源黑体正文；英文/公式用 Times 系或 STIX；学术气
- **版式特征**：页眉细线 + 章节号；图表带编号与来源（Figure 1 式）；引用页脚化；结构严格对应论文章节
- **图像倾向**：实验图/数据图/流程图为主，装饰图为零；图必有编号和说明文字
- **页面密度**：中高（学术场景容忍密度，但一页一个论证步骤）
- **标题语气**：论文小节式，"3.2 消融实验：移除注意力层的影响"式
- **image2 适配**：不适合（编号系统和公式渲染不可靠），默认模式 B
- **直接导出 PPT**：非常适合

## Style Brief（注入生成器）

学术报告风格：近白背景，深蓝灰宋体标题带章节编号，页眉细分隔线，所有图表带 Figure 编号与来源说明，引用以页脚小字呈现，无任何装饰性图形，蓝色仅用于引用和强调数据，整体严谨、克制、符合学术评审的审美预期。
