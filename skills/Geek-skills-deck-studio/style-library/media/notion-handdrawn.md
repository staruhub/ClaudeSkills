# Notion 手绘

- **一句话定位**：v2 时代的招牌风格——手绘线条 + 便签质感的知识图解
- **适用场景**：概念图解、知识分享组图、方法论传播图、Newsletter 配图化演示
- **不适用场景**：正式商务汇报、数据密集页、需要权威感的场合
- **色板**：`#FFFFFF`（纸白底）/ `#191919`（手绘线与文字）/ `#FFE58F`（便签黄）/ `#D9E8FC`（浅蓝块）/ `#FADBD8`（浅粉块）——马克笔质感淡彩
- **字体气质**：手写风中文字体（如霞鹜文楷）标题 + 简洁黑体正文；随性但可读
- **版式特征**：手绘框线与箭头连接概念；便签块承载要点；留白像笔记本页；小涂鸦图标
- **图像倾向**：全手绘插画系统——火柴人、简笔物件、波浪下划线；拒绝照片
- **页面密度**：中；一页一个概念及其关系
- **标题语气**：口语讲解式，"其实就三层"式
- **image2 适配**：非常适合（这个风格因 image 生成而生）
- **直接导出 PPT**：不推荐（手绘质感导出后失真），交付物是图片组

## Style Brief（注入生成器）

Notion 手绘知识图解风格：纸白背景，黑色手绘不规则线条画出框、箭头和分区，要点写在便签黄/浅蓝/浅粉的淡彩色块上，手写感中文标题，简笔涂鸦小图标和火柴人辅助说明，留白充足像一页整洁的笔记，整体亲切、清晰、有"聪明朋友给你画图讲明白"的感觉。
