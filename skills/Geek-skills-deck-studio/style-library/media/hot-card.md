# 热帖卡片

- **一句话定位**：小红书式高对比卡片组图，为信息流停留而生
- **适用场景**：小红书图文、知识卡片组图、自媒体复盘、课程宣发
- **不适用场景**：正式商务汇报、融资路演、需要打印的材料
- **色板**：`#FFF8F0`（奶油底）/ `#FF4D4D`（热点红）/ `#1A1A1A`（大字黑）/ `#FFD93D`（荧光黄高亮块）/ 每套可换一组主题色但保持三色内
- **字体气质**：超粗黑体大字 + 手写体点缀；标题即封面卖点
- **版式特征**：3:4 竖版优先；大字报式首图；荧光笔高亮关键词；序号卡片连载感；角标/贴纸元素
- **图像倾向**：贴纸、涂鸦箭头、emoji 风插画（注意：仅装饰，不做功能图标）；实拍图加白边框
- **页面密度**：单卡信息量小（1 观点 + 3 支撑），靠多卡成组
- **标题语气**：钩子式，"90% 的人第一步就错了"式（钩子要有真内容兜底，不做纯标题党）
- **image2 适配**：非常适合（本来就是图产品）；比例参数改 3:4
- **直接导出 PPT**：不适合导 pptx，交付物就是图片组

## Style Brief（注入生成器）

小红书爆款卡片风格：3:4 竖版，奶油色背景，超大加粗黑色标题字占画面三分之一，关键词用荧光黄色块高亮，红色用于序号与强调贴纸，涂鸦风箭头和下划线引导视线，单卡只讲一个观点配三条支撑，边缘留白充足，整体年轻、高对比、信息流里一眼停住。
