# Neubrutalism 硬糖

![签名页预览](neubrutalism-preview.png)

- **一句话定位**:黑描边 + 硬阴影 + 撞色的"新粗野"数字风,年轻而笃定
- **适用场景**:开发者产品、社区活动、自媒体、青年向品牌、工具类 App 介绍
- **不适用场景**:高管汇报、政企场合、奢侈品/高端服务
- **色板(封闭)**:`#FFE500` 黄 / `#FF6B9D` 粉 / `#00D4FF` 蓝 / `#BFFF00` 绿(每页主色选一)+ `#111111` 黑 / `#FFFDF5` 米白
- **字体气质**:超粗黑体标题;正文 Medium;数字用等宽
- **版式特征(CSS 配方级)**:所有卡片 `border: 3px solid #111` + `box-shadow: 8px 8px 0 0 #111`(硬阴影零模糊)+ `border-radius: 0`;元素可轻微旋转(±2°)制造手作感
- **图像倾向**:粗描边贴纸式图形;禁柔和渐变与羽化
- **页面密度**:中高(卡片天然容纳密度)
- **标题语气**:直给,"就这么干"式
- **image2 适配**:非常适合(描边+平涂是舒适区)
- **直接导出 PPT**:适合

## Style Brief(注入生成器)

Neubrutalism 风格:米白背景,内容装在 3px 纯黑描边、8px 硬投影(无模糊无透明)、直角的卡片里,单一高饱和主色(黄/粉/蓝/绿选一)大面积使用,超粗黑体标题,元素可 1-2 度轻微旋转,整体像一张自信的贴纸拼贴——粗粝、直接、绝不含糊。

> 依据:T6 评级「高」,当代数字风格中量化最高(NN|g 正式定义,自带 CSS 配方)。

**模板种子**:`neubrutalism-template.html`(已渲染验证的签名页,改文案即可复用其 token 与构图)。
