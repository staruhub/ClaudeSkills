# 黑白账本

- **一句话定位**：表格与数字为主角的数据汇报风，像一本印刷精良的账簿
- **适用场景**：财报/经营数据汇报、OKR 复盘、预算评审、周期性数据例会
- **不适用场景**：品牌故事、路演开场、情绪调动型内容
- **色板**：`#FFFFFF`（底）/ `#111111`（文字与表格线）/ `#F2F2F0`（隔行底纹）/ `#0B6E4F`（增长绿）/ `#B3261E`（下降红）——红绿只用于数据方向
- **字体气质**：正文思源黑体；所有数字与表格用等宽或 DIN 系字体，小数点对齐
- **版式特征**：表格是一等公民；细横线多、竖线少；大数字页（big-number）与表格页交替；页脚有数据口径注释
- **图像倾向**：几乎无图；图表只用柱状/折线/瀑布三种，拒绝 3D 和环形装饰
- **页面密度**：中高（数据场景允许密，但一页仍只回答一个问题）
- **标题语气**：口径精确的结论句，"营收同比 +18%，增速连续两季放缓"式
- **image2 适配**：不适合（表格渲染不可靠），此风格默认模式 B
- **直接导出 PPT**：非常适合

## Style Brief（注入生成器）

数据账本风格：纯白背景，黑色细线表格为版面主体，隔行浅灰底纹，数字用等宽字体右对齐，绿色仅表示增长、红色仅表示下降，图表限定柱状/折线/瀑布图且无任何 3D 或装饰效果，页脚小字注明数据口径，整体像一份严谨的印刷财报。
