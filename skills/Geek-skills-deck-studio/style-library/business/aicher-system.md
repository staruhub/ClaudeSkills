# Aicher 信息系统

![签名页预览](aicher-preview.png)

- **一句话定位**:慕尼黑 1972 奥运系统的当代应用——功能分色 + 网格图标,信息设计的黄金标准
- **适用场景**:多板块数据汇报、系统/平台介绍、组织架构与流程、指南手册化 deck
- **不适用场景**:情绪化路演、单一主题深潜(分色系统需要多板块才发挥)
- **色板(封闭,功能编码)**:浅蓝 `#5BA3D9`(主)/ 亮绿 `#7CB342` / 橙 `#F09235` / 银灰 `#9BA3AB` ——**每个内容域固定一色**(如:数据=蓝,流程=绿,风险=橙),全 deck 一致;底 `#FAFBFC` / 字 `#1F2429`
- **字体气质**:Univers 系(用 Helvetica Neue/Inter 代):中性、均匀;中文思源黑体 Medium
- **版式特征**:模块网格外露(可见细网格线);**像素网格图标**(人形/符号在正交+45°坐标系内构成);色带作导航
- **图像倾向**:网格化图标系统 > 照片;信息图表是主角
- **页面密度**:中高(信息系统天然多内容)
- **标题语气**:功能性陈述,"三个板块,一套系统"式
- **image2 适配**:适合
- **直接导出 PPT**:非常适合

## Style Brief(注入生成器)

Aicher 信息系统风格:近白背景带可见的细网格,内容按功能域分色编码(蓝/绿/橙,每域一色全篇一致),顶部彩色色带做章节导航,图标由正交与 45 度线段在网格内构成(奥运图标式人形/符号),Univers 气质的中性无衬线字,整体像 1972 慕尼黑奥运指南——系统的、彩虹的、绝对有序的。
