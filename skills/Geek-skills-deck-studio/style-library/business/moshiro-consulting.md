# 墨白咨询

- **一句话定位**：黑白灰 + 单强调色，麦肯锡式的克制与秩序
- **适用场景**：咨询报告、高管汇报、季度总结、尽调材料
- **不适用场景**：校园活动、自媒体传播、需要情绪感染力的路演
- **色板**：`#1A1A1A`（主文字）/ `#FFFFFF`（底）/ `#8C8C8C`（辅助）/ 强调色单选 `#C8102E` 或 `#0057B8`（全套只用一个）
- **字体气质**：思源黑体 Medium 标题 + Regular 正文；数字用 DIN；克制，无装饰字体
- **版式特征**：大量留白；标题写结论；页顶细线导航；图表灰阶为主、强调色只标关键数据
- **图像倾向**：少图；图表 > 照片；照片必须黑白或去饱和处理
- **页面密度**：中低；一页一个论点
- **标题语气**：陈述句结论，"Q3 获客成本下降 23%"式
- **image2 适配**：适合（版式简单、文字区域清晰）
- **直接导出 PPT**：适合（模式 B 首选风格）

## Style Brief（注入生成器）

极简咨询风格：纯白背景，深灰近黑文字，全套唯一强调色（红或蓝二选一），思源黑体，标题加粗左对齐写结论句，大面积留白，页面元素严格网格对齐，图表使用灰阶加单一强调色标注关键数据，无任何装饰性图形，专业、克制、可信。
