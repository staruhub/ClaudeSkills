# 极夜科技

- **一句话定位**：深底霓虹的 AI/数据产品气质，克制版赛博
- **适用场景**：AI 产品发布、技术架构讲解、数据平台演示、开发者大会
- **不适用场景**：传统行业客户提案、教学课件、印刷输出（深底费墨）
- **色板**：`#0A0E1A`（深蓝黑底）/ `#00D4FF`（青蓝主强调）/ `#7B61FF`（紫辅助，克制使用）/ `#E6EDF3`（文字）/ `#1C2333`（卡片底）
- **字体气质**：思源黑体 + 等宽字体（JetBrains Mono）点缀代码/数据；未来感但不花哨
- **版式特征**：深色卡片分区；细发光边线；数据大数字展示；架构图节点化
- **图像倾向**：抽象几何/粒子/网络拓扑 > 实物照片；渐变光效点到为止
- **页面密度**：中低；数据页可用 big-number 版式撑场
- **标题语气**：产品化短语，"毫秒级响应，百万级并发"式
- **image2 适配**：适合但需盯紧（霓虹元素容易过度，禁止项要加"装饰光效不超过两处"）
- **直接导出 PPT**：适合；注意投影仪深色显示效果需现场验证

## Style Brief（注入生成器）

深色科技风格：深蓝黑背景，浅色文字，青蓝色作主强调用于关键数据与标题下划线，深色圆角卡片分区带极细发光描边，配图为抽象几何、节点网络或粒子流，装饰光效每页不超过两处，等宽字体点缀技术元素，整体精密、未来、克制不赛博朋克。
