# 铂灰未来

- **一句话定位**：浅色系的未来感——铂灰金属质感，科技但明亮
- **适用场景**：ToB 科技产品提案、硬件发布、企业数字化方案、需要投影良好的科技场合
- **不适用场景**：需要暗场氛围的发布会（用极夜科技）、亲和型培训
- **色板**：`#F4F5F7`（铂灰底）/ `#15171A`（碳黑文字）/ `#6E7B8B`（金属灰辅助）/ `#0F62FE`（电光蓝强调）/ 渐变仅允许灰阶
- **字体气质**：思源黑体 + 无衬线英文（Inter/Roboto）；字距略宽，冷静精密
- **版式特征**：大面积浅灰留白；细金属质感分隔线；产品图悬浮投影；参数用规格表呈现
- **图像倾向**：产品渲染图、线框图、等距视角插画；避免人物摆拍
- **页面密度**：低中；参数页可密，叙事页必须疏
- **标题语气**：规格化陈述，"3 纳米制程，功耗降低 40%"式
- **image2 适配**：适合（浅底大留白是舒适区）
- **直接导出 PPT**：非常适合；投影表现优于深色系

## Style Brief（注入生成器）

铂灰科技风格：浅铂灰背景，碳黑文字，电光蓝仅用于关键参数与按钮式强调，细灰分隔线带金属质感，产品图悬浮并带柔和投影，等距线框插画表达架构，灰阶渐变之外无其他渐变，整体明亮、精密、有旗舰硬件发布会的质感。
