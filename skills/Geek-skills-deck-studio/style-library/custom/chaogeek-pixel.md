# ChaoGeek 像素半调

- **一句话定位**：ChaoGeek 品牌专属，像素+半调质感的极客风，DNA 铁律不可协商
- **适用场景**：ChaoGeek 公众号配图化演示、超极客课程/分享、Pong 个人品牌内容
- **不适用场景**：非 ChaoGeek 品牌的任何内容（这是品牌资产，不外借）
- **色板（DNA 铁律）**：霓虹绿 `#39FF14` / 龙虾红 `#FF3D2E` / 深底 `#0D0D12`；三色之外只允许白与灰阶
- **字体气质（DNA 铁律）**：LOGO 与英文点缀用 Press Start 2P；中文标题思源黑体 Heavy 超粗；正文思源黑体 Regular
- **版式特征**：像素网格/扫描线/半调网点三种质感至少用其一；标题可加像素风描边；机械龙虾元素可作点题视觉
- **图像倾向**：像素画、半调处理照片、等距工程图风格插画；禁止圆润可爱风
- **页面密度**：中低；大标题主导
- **标题语气**：极客宣言式，带梗但有信息量
- **image2 适配**：适合；prompt 中三色色值必须原样给出，并声明"不得引入其他饱和色"
- **直接导出 PPT**：适合；Press Start 2P 需嵌入或替换为图片

## Style Brief（注入生成器）

ChaoGeek 品牌极客风格：深色 `#0D0D12` 背景带扫描线或半调网点质感，霓虹绿 `#39FF14` 作主强调、龙虾红 `#FF3D2E` 作次强调，三色之外仅允许白与灰，中文超粗黑体大标题，英文点缀像素字体，可出现像素风机械龙虾元素，整体硬核、复古未来、辨识度极强，不得引入紫色渐变等任何外部配色。
