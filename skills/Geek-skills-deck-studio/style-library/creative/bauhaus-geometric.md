# Bauhaus 几何

![签名页预览](bauhaus-preview.png)

- **一句话定位**:三原色 + 圆方三角的构成游戏,现代设计的源头味
- **适用场景**:设计/创意行业分享、教育创新、艺术活动、方法论发布
- **不适用场景**:严肃财报、政府汇报、需要柔和亲和的内容
- **色板(封闭)**:`#E53935` 红 / `#FDD835` 黄 / `#1E88E5` 蓝 + `#111111` 黑 / `#F7F4EF` 暖白 / `#8A8A8A` 灰——三原色每页至多两色同时出现
- **字体气质**:思源黑体 Heavy 大标题(致敬 Bayer 几何字);正文 Regular;可全小写英文点缀
- **版式特征**:**平衡的不对称**;几何形(圆/方/三角,禁其他形状)既是装饰也是构图骨架;粗黑分割线
- **图像倾向**:纯几何 SVG 构成,拒绝照片;图表用几何形重构
- **页面密度**:中低;形与字互为主角
- **标题语气**:宣言式,"设计不是装饰"式
- **image2 适配**:适合(纯几何 CSS/SVG 即可,无需图像模型)
- **直接导出 PPT**:适合

## Style Brief(注入生成器)

包豪斯构成风格:暖白背景,黑色粗字标题,红黄蓝三原色几何形(仅圆/方/三角)作为构图主体而非装饰,平衡的不对称布局,粗黑水平分割线,几何形可与文字重叠制造张力,整体像 1923 年魏玛展览海报的当代复刻——原色的、几何的、宣言式的。

> 依据:T6 普查评级「高」(可量化/约束/普适/渲染/母题全高);母题限定圆方三角是硬约束。

**模板种子**:`bauhaus-template.html`(已渲染验证的签名页,改文案即可复用其 token 与构图)。
