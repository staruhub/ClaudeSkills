# 英黄工作室

- **一句话定位**：黑金气质的高端提案感，大字、留白、强标题
- **适用场景**：策略提案、品牌方案、咨询汇报、高客单价销售
- **不适用场景**：教学课件、政府汇报、数据密集型财报
- **色板**：`#0D0D0D`（深黑底）/ `#E8B004`（暖黄强调）/ `#F5F0E6`（米白文字）/ `#3D3D3D`（分隔）
- **字体气质**：超粗黑体大标题（思源黑体 Heavy）+ 细体正文；中英混排时英文用 Montserrat
- **版式特征**：大字、低密度、强标题主导；暗底上的黄色关键词高亮；不对称留白构图
- **图像倾向**：少图；大面积暗底 + 单点题视觉（一个物体/一个隐喻图像）；照片压暗处理
- **页面密度**：低；一页一句话一个视觉
- **标题语气**：宣言式短句，"把复杂留给我们"式
- **image2 适配**：非常适合（低密度大字是 image 模型的舒适区）
- **直接导出 PPT**：适合，但需嵌入字体

## Style Brief（注入生成器）

高端黑金提案风格：近黑深色背景，米白色超粗黑体大标题占据视觉中心，暖黄色只用于关键词高亮和细线装饰，页面元素稀疏、留白大胆不对称，配图为单一压暗的点题视觉物，整体气质昂贵、笃定、有提案说服力。
