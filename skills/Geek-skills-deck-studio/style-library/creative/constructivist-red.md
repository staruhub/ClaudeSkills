# 构成主义红

![签名页预览](constructivist-preview.png)

- **一句话定位**:红黑白 + 对角线的能量构图,变革与冲锋的视觉语言
- **适用场景**:产品发布"宣战"页、增长/变革主题、活动动员、竞品对垒叙事
- **不适用场景**:平和的教学内容、需要中立感的报告;注意历史宣传联想,政企场合慎用
- **色板(封闭)**:`#D32F2F` 革命红 / `#111111` 黑 / `#F5F1E8` 纸白——三色,偶许 `#E8B004` 黄点缀(≤1 处/页)
- **字体气质**:超粗黑体(800-900)大标题,允许沿对角线排布;超大标点符号(!?→)可作视觉元素
- **版式特征**:**对角线构图是签名**(文字/色块/线条沿 10-20° 斜线);红楔入白圆的碰撞范式;照片可做单色蒙太奇
- **图像倾向**:几何碰撞 > 图像;若用照片必须红/黑双色调处理
- **页面密度**:中;张力靠角度不靠数量
- **标题语气**:动员式短句,"打破它"式
- **image2 适配**:适合;prompt 必须声明对角线构图与三色限制
- **直接导出 PPT**:适合

## Style Brief(注入生成器)

俄国构成主义风格:纸白背景,革命红与纯黑双主色,标题文字沿对角线排布,红色楔形/三角与黑色圆形碰撞构图,超大标点符号作为图形元素,粗重无衬线字,整体有 El Lissitzky《红楔入白圆》的运动能量——倾斜的、碰撞的、宣言式的。

> 依据:T6 评级「高」;Paula Scher 的 Public Theater 已证明此语言可移植当代商业内容。

**模板种子**:`constructivist-template.html`(已渲染验证的签名页,改文案即可复用其 token 与构图)。
