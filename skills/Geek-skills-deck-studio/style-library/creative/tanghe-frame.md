# 糖盒彩框

- **一句话定位**：低饱和奶油底 + 彩色块与边框标签，新潮杂志感
- **适用场景**：创意提案、品牌介绍、校园/活动策划、青年向产品
- **不适用场景**：严肃财报、政府汇报、高管例会
- **色板**：`#FAF6EF`（奶油底）/ `#F4A261`（杏橙）/ `#8AB6D6`（雾蓝）/ `#B5C99A`（豆绿）/ `#2B2B2B`（文字）——彩色只做色块与边框，不做文字色
- **字体气质**：圆润黑体标题（阿里巴巴普惠体 Bold）+ 常规正文；允许手写体标签点缀
- **版式特征**：内容装在带圆角边框的"糖盒"里；角落贴纸式标签；多区块拼贴；页码做成小徽章
- **图像倾向**：插画与实拍混搭但统一加奶油色调滤镜；装饰性小元素（星星/波浪线）克制使用
- **页面密度**：中；区块多但每块内容少
- **标题语气**：轻快短语，"先看三件小事"式
- **image2 适配**：非常适合（装饰系统 image 模型表现好）
- **直接导出 PPT**：适合，但边框系统需模板化

## Style Brief（注入生成器）

奶油杂志拼贴风格：低饱和奶油色背景，内容装在圆角细边框色块中，杏橙/雾蓝/豆绿三色轮换做区块底色，深灰文字，角落有贴纸式小标签和徽章页码，少量星星波浪线装饰但不盖过内容，整体新潮、轻盈、有杂志内页感。
