# Visual Checklist — 渲染后逐条过检（L2 视觉门禁）

> 用法：模式 B/C 渲染出 PNG 后，**亲眼看图**逐条判定；违例定位到 token / layout 参数修复后重渲染（最多 2 轮）。
> 来源两层：A 组来自设计原典（带出处），B 组来自本 skill 三轮实战盲评的教训。

## A. 原典量化规则（可判定）

| # | 规则 | 来源 |
|---|------|------|
| A1 | 每页 1 个观点；3 秒内可理解（读完标题+扫一眼即懂） | Duarte glance test |
| A2 | 每页文字 ≤75 词；超过即"文档化"，拆页或改图 | Duarte slide:ology |
| A3 | 不用 bullet 堆叠 ≥3 条；改图示/时间线/图标+短语 | Reynolds |
| A4 | 全页配色 ≤2–3 色；大图 full-bleed 优先于多张小图 | Reynolds |
| A5 | 图表 Y 轴不截断（Lie Factor 0.95–1.05）；3D 效果/背景纹理/冗余网格线与图例 = 0 | Tufte |
| A6 | 非数据墨水最少化：删除后信息不变的元素必须删 | Tufte data-ink |
| A7 | 所有元素对齐统一网格，占整数个 grid field；gutter 与页边距全篇一致 | Müller-Brockmann |
| A8 | 西文行长 45–90 字符 / 7–10 词每行 | Butterick / M-B |
| A9 | 行距：西文 = 字号 120–145%；中文行间距 = 字号 50–100%（口径：行间空白/字号） | Butterick / clreq |
| A10 | 全大写文本加 5–12% 字距，且不超过一行 | Butterick |
| A11 | 中西文混排留 1/4 汉字宽空隙（行首行尾除外） | W3C clreq |
| A12 | 行首无点号/结束引号；行尾无开始引号/括号（禁则零违例）；中文正文标点全角 | W3C clreq |

## B. 实战教训规则（三轮盲评 + 独立评审沉淀）

| # | 规则 | 出处（教训） |
|---|------|-------------|
| B1 | **重锚点必须存在**：每页至少一个粗黑标题或数据英雄；细体（200-300）只做对比层 | v4 全盘细体盲评 3:0 落败 |
| B2 | 每页有**视觉概念**（图形讲故事），不是文本框排列；数据页图形自身必须承载叙事 | 评审"唯一内容即视觉的是 p7" |
| B3 | 卡片内部三段网格（标题/内容层/结论行），**禁止"头重-空腹-脚轻"** | deck5-p3 / deck6-p4 两次复发 |
| B4 | 垂直重心经营：内容占版心 60–75%，禁止下半页 ≥40% 真空 | deck6-p7 评审点名 |
| B5 | 明暗节奏：连续同主题 ≤3 页；9 页 deck 至少 1–2 个反转页 | 归藏节奏规则 + deck6-p4 被评最强 |
| B6 | 一个贯穿母题（幽灵字/同心环/节点网格/大圆），封面即确立，风格族内复用 | 评审"缺图形母题"3 次 |
| B7 | 强调色克制：单页强调元素 ≤3 处；发光/阴影 ≤2 处；整列涂色 = 强调失效 | v2 对比页教训 |
| B8 | 对位对称：并列元素结构必须同构（都有顶线/都实心）；"有 vs 无"读作遗漏 | deck6-p7 空心节点被误读 |
| B9 | 幽灵大字与正文标题不得碰撞裁切；出血要么彻底要么不出血 | deck4 封面首版翻车 |
| B10 | 任取两页抽查：骨架（页眉/页脚/边距/字阶）逐像素一致 | 一套 deck 一个风格铁律 |

## C. 判定流程

1. 渲染全部页 → 通读一遍缩略,先看**节奏**(B5)与**母题**(B6)
2. 抽 3 页细看:封面、数据页、最密的一页 → 过 A 组量化项与 B1-B4
3. 违例 → 定位到 token(色/字/距)或 layout(结构)→ 修 → 只重渲染受影响页
4. 第 2 轮仍违例的项如实交付并声明,不隐瞒
