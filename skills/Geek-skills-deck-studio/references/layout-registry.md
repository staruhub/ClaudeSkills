# Layout Registry — 版式注册表 v1

> 借鉴归藏 S01–S22 与 Slidev named-layouts 的"锁定版式"范式：**每页从注册表选版式并声明
> `layout: LXX`，不从零发明**。留白与构图设计在版式里，不是内容没填满剩下的。
> 全部版式来自三套已验证样例（examples/），每条含硬规则与实证出处。
> 不在册的版式需先在此注册（结构 + 硬规则 + 实证）再使用。

## 使用规则

1. **选版式在写内容之前**（Step 5）：先按页面类型（page-schema 12 型）从本表选 layout，再填内容
2. 每页产出物（HTML 注释或 brief）声明 `layout: LXX`，便于校验与回归
3. 版式内的**结构性留白/网格/字阶不许改**；可改的只有内容与 token 引用
4. 新场景装不进任何在册版式 → 先注册新版式（含硬规则+至少一次渲染实证）再用——禁止一次性野版式

## 正交拆维（组织原则，来自竞品研究）

风格 = **视觉流派**（style-library，管色/字/质感）× **版式骨架**（本表，管构图）× **主题色**（token，封闭集合）。
三者独立策展、自由组合：同一 L05 槽位叙事可渲染成墨白/极夜/英黄三种质感。
风格用流派专名命名（墨白咨询/极夜科技），场景词只做 scene-routing 的检索入口——不要按场景造风格。
