# 信息图风格指南 (Mode B)

本文件定义 Mode B (信息图提示词) 的完整风格规范。
每张图的提示词必须包含完整的风格前缀和后缀，不可省略。

## 提示词组装公式

```
{风格前缀 — 复制粘贴，不改一字}

{内容描述 — 唯一变量区域}

{风格后缀 — 复制粘贴，不改一字}
```

---

## 风格前缀/后缀完整版

### 风格 1: Notion 手绘风 (默认)

**前缀:**
Notion官方插画风格松弛感线稿信息图。线条特征：纯黄色为主，搭配其他适合的颜色。线条粗细不均匀，像马克笔随手画的质感。笔触松弛、略带抖动，不追求工整。所有图标、人物、图表都保持手绘涂鸦感。少部份地方填充黑色让整体压实。人物画法：简笔画风格女孩，短发，圆润的头、点或线表示五官，肢体夸张、动作松弛自然，可以是完整人物或只露出局部。

**后缀:**
禁止事项：不要彩色渐变或复杂配色。不要粗黑边框或生硬分隔线。不要3D效果、阴影、立体感。不要密集文字堆砌，文字内容简化。保持大量留白和呼吸感。装饰元素要克制。图片比例16:9，中文标注。

---

### 风格 2: 极简商务风

**前缀:**
极简主义商务信息图。纯色背景（浅灰#f8f9fa或纯白），使用几何形状分区。扁平化图标，线条简洁利落。配色严格限制在3色以内：一个主色、一个辅助色、一个灰色。字体清晰现代（无衬线体）。信息密度低，每个区域只放一个核心信息。

**后缀:**
禁止事项：不要手绘风格或涂鸦。不要超过3种颜色。不要渐变、阴影或3D效果。不要密集排版。保持50%以上留白。不要装饰性元素（花边、边框装饰等）。图片比例16:9，中文标注。

---

### 风格 3: 深色科技风

**前缀:**
深色主题科技信息图。背景使用深蓝(#0a192f)或纯黑(#0d1117)。内容使用霓虹色高亮：青色(#00d4ff)、绿色(#00ff88)、橙色(#ff8800)。线条纤细(1-2px)并带有微弱发光效果。数据以发光面板/卡片形式展示，卡片背景半透明。图标为线性图标。整体氛围：未来感、专业、高端科技。

**后缀:**
禁止事项：不要亮色背景。不要手绘风格。不要超过2种霓虹强调色。不要过多文字，用数据和图形说话。保持暗部层次感。不要密集排版。图片比例16:9，中文标注。

---

### 风格 4: 莫兰迪柔和风

**前缀:**
莫兰迪色系柔和信息图。低饱和度配色：雾粉(#edafb8)、灰绿(#b0c4b1)、淡紫(#cdb4db)、米白(#f7e1d7)、温灰(#dedbd2)。元素带有柔和投影(offset 2px, blur 8px, 透明度20%)。排版优雅，行距宽松(1.6-1.8倍)。圆角卡片(border-radius 12-16px)。适合高端品牌和女性受众。整体氛围：精致、宁静、高级感。

**后缀:**
禁止事项：不要高饱和度颜色。不要直角或生硬边框。不要粗体文字（标题除外）。不要密集信息。保持大量呼吸空间。装饰元素限于柔和的几何形状。图片比例16:9，中文标注。

## 不同图片类型的内容描述模板

### 封面图
```
画面中央大标题"{主题名}"，旁边一个{人物动作描述}。
周围散落着与主题相关的{图标元素}。
底部小字"{副标题/slogan}"。
```

### 数据冲击图
```
画面中央一个巨大的"{关键数字}"，用{颜色}色块衬底。
一个简笔画人物{反应动作}看着这个数字。
旁边小字标注"{数据含义}"，底部来源标注"{来源 年份}"。
```

### 对比图
```
画面分为左右两部分。左边是{旧状态/问题}，右边是{新状态/方案}。
中间用箭头或分隔表示转变。
{人物在两边的不同表情/动作}。底部数据"{关键对比数据}"。
```

### 流程/时间轴图
```
从{方向}的{N}个步骤/节点，用{连接方式}连接。
每个节点用一个{图标/人物动作}表示。
标注每个步骤的关键词：{步骤1}→{步骤2}→...
```

### 总结/CTA图
```
一个自信的人物站在{成果/目标}旁边，做出{胜利/邀请}的手势。
周围环绕{核心要点}的关键词。
底部大字"{行动号召语}"。
```

## 多图一致性保障

### 硬性约束（不可修改）

| 约束项 | 规范 |
|-------|------|
| 风格前缀 | 每张完整包含，一字不改 |
| 风格后缀 | 每张完整包含，一字不改 |
| 比例 | 16:9 |
| 语言 | 中文标注 |
| 同组风格 | 全部使用同一风格，不混搭 |

### 内容差异化

仅通过 [内容描述] 部分体现每张图的独特信息。
不要在风格描述中加入额外的颜色或效果。

## 输出文件命名

```
infographic-01.png — 封面
infographic-02.png — 内容图1
infographic-03.png — 内容图2
...
infographic-{N}.png — 总结图
```
