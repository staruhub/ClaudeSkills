# 省级自然科学基金报告格式规范

## 一、页面设置

### 1.1 纸张规格
- 纸张大小：A4（210mm × 297mm）
- 页边距：上2.5cm、下2.5cm、左2.8cm、右2.5cm
- 装订线：0.5cm（左侧）

### 1.2 页眉页脚
- 页眉：报告标题，宋体小五号，居中
- 页脚：页码，Times New Roman五号，居中
- 页眉页脚距边界：1.5cm

## 二、标题格式

### 2.1 报告标题
- 字体：黑体
- 字号：小二号
- 对齐：居中
- 段前段后：各0.5行

### 2.2 一级标题
- 格式：一、二、三、...
- 字体：黑体
- 字号：四号，加粗
- 对齐：左对齐
- 段前：0.5行，段后：0.5行

### 2.3 二级标题
- 格式：（一）（二）（三）...
- 字体：黑体
- 字号：小四号
- 对齐：左对齐
- 段前：0行，段后：0行

### 2.4 三级标题
- 格式：1. 2. 3. ...
- 字体：黑体
- 字号：小四号
- 对齐：左对齐，首行缩进2字符

### 2.5 四级标题
- 格式：(1) (2) (3) ...
- 字体：宋体
- 字号：小四号
- 对齐：首行缩进2字符

### 2.6 五级标题
- 格式：① ② ③ ...
- 字体：宋体
- 字号：小四号
- 对齐：首行缩进2字符

## 三、正文格式

### 3.1 中文正文
- 字体：宋体
- 字号：小四号
- 首行缩进：2字符
- 行距：固定值26-28磅
- 段前段后：0行

### 3.2 英文与数字
- 字体：Times New Roman
- 字号：小四号
- 与中文混排时保持一致

### 3.3 特殊标记
- 强调：加粗或使用引号
- 术语首次出现：全称（缩写）
- 外文专有名词：保持原文

## 四、图表格式

### 4.1 图片规范
- 标题位置：图片下方
- 标题格式：图X [空格] 图题（宋体五号，居中）
- 图片分辨率：≥300dpi
- 图片宽度：不超过版心宽度
- 段前段后：各0.5行

**示例**：
```
[图片]
图1 风速预报模型技术路线图
```

### 4.2 表格规范
- 标题位置：表格上方
- 标题格式：表X [空格] 表题（宋体五号，居中）
- 表格样式：三线表（上下粗线，表头下细线）
- 表格内文字：宋体五号，居中
- 段前段后：各0.5行

**示例**：
```
表1 不同模型预报性能对比
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
模型        MAE    RMSE    CC
────────────────────────────
LSTM        2.34   3.12   0.89
Transformer 2.01   2.87   0.92
本文模型    1.78   2.45   0.94
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

### 4.3 公式规范
- 独立成行的公式居中
- 公式编号：右对齐，格式(1)、(2)...
- 公式与正文间距：段前段后各0.5行
- 公式内变量：斜体

**示例**：
```
                    RMSE = √(1/n ∑(yᵢ - ŷᵢ)²)                    (1)
```

## 五、参考文献格式

### 5.1 GB/T 7714-2015 标准

**期刊论文**：
```
[序号] 作者. 文题[J]. 期刊名, 年, 卷(期): 起止页码.
```
示例：
```
[1] 张三, 李四. 基于深度学习的短时降水预报方法[J]. 气象学报, 2023, 81(3): 456-468.
```

**专著**：
```
[序号] 作者. 书名[M]. 版次. 出版地: 出版社, 年: 引用页码.
```
示例：
```
[2] 王五. 机器学习基础与应用[M]. 北京: 科学出版社, 2022: 123-145.
```

**学位论文**：
```
[序号] 作者. 论文名[D]. 学位授予单位所在地: 学位授予单位, 年.
```
示例：
```
[3] 赵六. 单站气温预报方法研究[D]. 南京: 南京信息工程大学, 2023.
```

**网络文献**：
```
[序号] 作者. 文题[EB/OL]. (发布日期)[引用日期]. URL.
```

### 5.2 引用标注
- 正文中引用格式：上标[1]或行内[1]
- 多篇引用：[1-3]或[1,3,5]
- 连续引用：[1-5]

## 六、摘要与关键词格式

### 6.1 中文摘要
- "摘要"二字：黑体小四号，顶格
- 摘要内容：宋体小四号，首行缩进2字符
- 字数：300-400字

### 6.2 关键词
- "关键词"三字：黑体小四号，顶格
- 关键词内容：宋体小四号
- 各关键词用分号（；）分隔
- 数量：3-5个

### 6.3 英文摘要
- "Abstract"：Times New Roman小四号加粗
- 摘要内容：Times New Roman小四号
- 首行不缩进

### 6.4 英文关键词
- "Keywords"：Times New Roman小四号加粗
- 关键词用分号分隔
- 首字母小写（专有名词除外）

## 七、目录格式

### 7.1 目录标题
- "目录"二字：黑体三号，居中
- 段后：1行

### 7.2 目录内容
- 一级标题：黑体小四号
- 二级标题：宋体小四号，缩进2字符
- 三级标题：宋体小四号，缩进4字符
- 行距：1.5倍
- 页码右对齐，用点线连接

## 八、附录格式

### 8.1 附录标题
- 格式：附录A、附录B、附录C...
- 字体：黑体四号
- 对齐：左对齐

### 8.2 附录内容
- 格式同正文
- 图表编号：图A-1、表B-1
