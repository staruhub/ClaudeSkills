# 考点分析指南

## 核心原则

> **考试考的不是"什么重要"，而是"老师认为什么重要"**

材料中的信息密度、强调方式、练习题分布，都是判断考点的重要信号。

## 信号词识别

### 高优先级信号词（几乎必考）

| 信号词 | 含义 |
|-------|------|
| "重点"、"重中之重" | 明确提示 |
| "必须掌握"、"必背" | 强制要求 |
| "考试常考"、"历年真题" | 直接关联考试 |
| "注意"、"特别注意" | 强调易错点 |
| "总结"、"归纳" | 浓缩精华 |
| 红色/加粗/下划线文字 | 视觉强调 |

### 中优先级信号词（大概率考）

| 信号词 | 含义 |
|-------|------|
| "区别"、"比较"、"异同" | 对比类题目素材 |
| "原因"、"意义"、"影响" | 简答题素材 |
| "步骤"、"流程"、"过程" | 过程分析题素材 |
| "例如"、"比如" | 例题可能变形 |
| "公式"、"定理"、"法则" | 计算/证明题基础 |

### 结构性信号（隐含重点）

| 结构特征 | 判断依据 |
|---------|---------|
| PPT页数多的章节 | 讲得多 = 老师重视 |
| 有专门练习题的知识点 | 练习 = 会考 |
| 章末总结中列出的内容 | 总结 = 精华 |
| 多次出现的概念 | 重复 = 强调 |

## 材料类型分析策略

### PPT分析法

1. **统计各章节页数**
   - 页数多 → 重点章节
   - 某知识点反复出现多页 → 核心考点

2. **识别PPT结构**
   ```
   标题页 → 章节重点预览
   内容页 → 具体知识点
   总结页 → 核心精华
   练习页 → 考试题型预示
   ```

3. **注意动画/强调**
   - 分步显示的内容通常是重点
   - 有动画效果的图表要特别关注

### 课本分析法

1. **章节导读/学习目标**
   - 直接告诉你要学什么
   - 对应考试的考查目标

2. **正文中的特殊标记**
   ```
   [定义] → 填空/判断题
   [定理] → 证明/应用题
   [例题] → 原题变形或考法参考
   [思考题] → 可能的大题
   [小结] → 必须掌握的内容
   ```

3. **章末习题分布**
   - 基础题 → 送分题范围
   - 综合题 → 大题素材
   - 开放题 → 论述题可能

### 考纲/复习范围分析法

1. **直接按条目整理考点**
   - 考纲说的 = 一定会考
   - 考纲没说的 = 几乎不考

2. **注意考纲的表述**
   ```
   "了解" → 选择/判断题
   "理解" → 简答题
   "掌握" → 计算/应用题
   "运用/分析" → 综合大题
   ```

### 往年真题分析法（如有）

1. **统计题型分布**
   - 选择题考什么？
   - 大题通常怎么出？

2. **识别高频考点**
   - 连续多年都考 = 必考
   - 隔年考一次 = 重要
   - 只考过一次 = 可能轮到

3. **分析难度梯度**
   - 第一题通常送分
   - 最后一题通常压轴

分析完材料后，输出结构化的考点清单：

```markdown

- 材料来源：[PPT/课本/考纲等]
- 分析日期：[日期]

---

### 🔴 高优先级考点（必考，重点突破）

#### 1. [考点名称]

- **考查方式**：选择题 / 填空题 / 简答题 / 计算题 / 案例分析
- **核心内容**：
  - [要点1]
  - [要点2]
- **老师强调**："[原话或标注]"
- **可能的陷阱**：[易错点]

#### 2. [考点名称]
...

---

### 🟡 中优先级考点（大概率考，需要掌握）

#### 1. [考点名称]
- **来源**：...
- **考查方式**：...
- **核心内容**：...

---

### 🟢 低优先级考点（可能考，有余力再看）

#### 1. [考点名称]
...
```

## 实战示例

### 输入：数据结构PPT目录

```
第1章 绑论（8页）
第2章 线性表（24页）
第3章 栈和队列（18页）
第4章 树（32页）
第5章 图（28页）
第6章 查找（16页）
第7章 排序（22页）
```

### 分析输出

```
页数分布分析：
- 树（32页）> 图（28页）> 线性表（24页）> 排序（22页）
- 预判：树和图是大重点

重点预测：
🔴 高优先级：
1. 二叉树的遍历（树章节核心）
2. 图的遍历和最短路径（图章节核心）
3. 各种排序算法比较（排序章节必考）

🟡 中优先级：
1. 链表操作（线性表应用）
2. 栈的应用（表达式求值等）
3. 二分查找（查找章节经典）

🟢 低优先级：
1. 绑论（页数最少，多为概念）
```

## 快速检查清单

分析材料时，逐项检查：

- [ ] 统计了各章节页数/篇幅
- [ ] 识别了所有信号词
- [ ] 找到了老师的明确提示
- [ ] 分析了练习题/例题分布
- [ ] 对比了考纲要求
- [ ] 整理了知识点层级关系
- [ ] 预判了可能的题型
- [ ] 标记了易错点和陷阱
