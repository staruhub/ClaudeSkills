# Coding Prompts

编程和开发相关的提示词模板集合。

## 📋 可用提示词

### 代码生成
- [code-review.md](./code-review.md) - 代码审查
- [bug-fix.md](./bug-fix.md) - Bug 修复
- [refactoring.md](./refactoring.md) - 代码重构
- [documentation.md](./documentation.md) - 代码文档生成
- [unit-tests.md](./unit-tests.md) - 单元测试生成

## 🎯 使用场景

- 代码质量审查
- 自动化测试编写
- 代码优化和重构
- 技术文档生成
- Bug 诊断和修复

## 💡 最佳实践

1. 提供完整的代码上下文
2. 说明使用的编程语言和框架
3. 描述具体的需求或问题
4. 包含相关的错误信息
5. 指定代码风格偏好
